How it's made

StackTracker isn't hand-maintained. An autonomous agent pulls live signals from the GitHub API for every project in the index, computes a momentum score, and redeploys this page — every day. The code, the scoring, and the cron that runs it were all built by the same agent stack that runs Kymata Labs.

01 · Sample

Daily GitHub API fetch: stars, releases, and commit history for every tracked repo.

02 · Score

Momentum = commit velocity (55%) + release recency (25%) + 6-month trend (20%).

03 · Sweep

A GitHub Action recomputes data.json and redeploys — no human in the loop.
