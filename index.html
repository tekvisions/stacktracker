About StackTracker

StackTracker is the live momentum index for AI infrastructure — the serving engines, orchestration frameworks, vector stores and ops tooling that run AI in production. Every tool is ranked daily from live GitHub signals, so the index reflects which infrastructure the ecosystem is actually building on right now. It is one of The Living Indexes, a fleet built and operated end-to-end by Kymata Labs' AI agents.

What is StackTracker?

StackTracker is a live momentum index for AI-infrastructure tools — inference servers, orchestration frameworks, vector databases and ops tooling — ranked by GitHub momentum and recomputed daily. It surfaces the infrastructure the ecosystem is adopting now. It is one of The Living Indexes by Kymata Labs.

How is momentum scored?

A 0–100 score blending log-scaled GitHub stars (55%), push-recency (32%, decaying to zero by about 180 days) and rising-newness (13%). A tool that shipped this week can outrank a larger tool that has gone quiet — momentum, not legacy.

How often is it updated?

Every day. A GitHub Action recomputes each tool's momentum from live GitHub signals and republishes automatically, with no human in the loop.

How it's made

A self-updating intelligence instrument, built and run by an AI agent.

StackTracker isn't hand-maintained. An autonomous agent pulls live signals from the GitHub API for every project in the index, computes a momentum score, and redeploys this page — every day. The code, the scoring, and the cron that runs it were all built by the same agent stack that runs Kymata Labs.
